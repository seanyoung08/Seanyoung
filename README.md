# Seanyoung
# 51 智能小车

## 项目简介

本项目是一个基于 **STC89C52RC 单片机** 的 51 智能小车系统，主要实现蓝牙遥控、超声波避障、红外循迹和 OLED 状态显示等功能。项目通过多个传感器采集环境信息，并结合 PID 算法对小车循迹过程中的偏移误差进行修正，使小车能够完成较稳定的自动行驶。

项目适合作为单片机课程设计、嵌入式入门项目或硬件开发实习简历项目展示。

## 项目功能

- **蓝牙遥控**
  - 通过蓝牙模块接收手机端发送的控制指令
  - 使用 UART 串口通信控制小车前进、后退、左转、右转和停止
  - 在 OLED 屏幕上实时显示当前运动状态

- **超声波避障**
  - 使用 HC-SR04 超声波模块进行距离检测
  - 通过 GPIO 触发测距并读取回响信号
  - 设置 30cm 安全阈值，当检测到障碍物时自动转向避障

- **红外循迹**
  - 使用 5 个红外反射光电传感器识别黑线轨迹
  - 根据不同传感器组合判断小车偏移方向
  - 结合 PID 算法修正运动误差，提高循迹稳定性

- **OLED 状态显示**
  - 使用 IIC 协议驱动 OLED 显示模块
  - 显示当前模式、运动状态、避障状态等信息

## 软件设计

### 1. 蓝牙遥控模块

蓝牙模块通过 UART 与 STC89C52RC 通信。手机端发送控制字符后，单片机在串口中断或轮询中读取指令，并根据不同指令控制小车运动。

常见指令示例：

| 指令 | 功能 |
| --- | --- |
| `F` | 前进 |
| `B` | 后退 |
| `L` | 左转 |
| `R` | 右转 |
| `S` | 停止 |

### 2. 超声波避障模块

HC-SR04 通过 Trig 引脚触发测距，通过 Echo 引脚返回高电平持续时间。程序根据回响时间计算距离：

```text
距离 = 高电平持续时间 * 声速 / 2
```

当检测距离小于 30cm 时，小车进入避障逻辑，停止当前前进动作并执行转向。

### 3. 红外循迹模块

小车底部安装 5 路红外反射光电传感器，通过读取不同传感器的状态判断小车相对黑线的位置。

程序根据传感器组合计算偏差值，并通过 PID 算法对左右电机速度进行修正：

```text
输出 = Kp * 当前偏差 + Ki * 偏差累积 + Kd * 偏差变化量
```

通过 PID 修正后，小车能够减少左右摇摆，提高循迹稳定性。

### 4. OLED 显示模块

OLED 使用 IIC 协议与主控通信，主要用于显示当前运行模式、蓝牙指令状态、避障距离或循迹状态，方便调试和观察小车运行情况。

## 使用方法

1. 按照硬件连接说明完成 STC89C52RC、蓝牙模块、超声波模块、红外循迹模块、电机驱动和 OLED 的接线。
2. 使用 Keil 打开项目工程，编译生成 HEX 文件。
3. 使用 STC-ISP 将程序下载到 STC89C52RC 单片机。
4. 打开手机蓝牙调试助手，连接蓝牙模块。
5. 发送控制指令，测试小车前进、后退、转向和停止。
6. 切换到自动模式，测试超声波避障和红外循迹功能。
